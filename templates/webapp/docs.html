Швидкий старт

Для того, щоб почати роботу з годинниками, треба спочатку хоча б один. Є два способи це зробити:

  • Панель керування
  • API

Створимо його через панель керування:

  1. Заходимо в особистий кабінет нажавши Привіт, <USERNAME>;
  2. Вибираємо на боковій панелі Віртуальні годинники
  3. Перейшовши на сторінку, нажимаємо кнопку <+ створити>;
  4. Іменуємо наш годинник(опціонально), та нажимаємо кнопку <Створити>.
  5. Нас повертає на сторінку віртуальних годинників і ми бачимо новоствроренний годинник, нажимаємо на його назву (або None), та переходимо на сторінку налаштування останнього.
  6. Перейшовиши в налаштування годинника, можемо робити небхідні з ним налаштування, а потім зберігаємо їх.
  7. Щоб отримати значення годинника через АПІ, переходимо назад, та нажимаємо кнопку Посилання на годинник.
  8. Використовуючи клієнтський код, виконуйте get запити до https://{{request.get_host}}/api/v1/clocks/<clock_id> додаючи заголовок "Authorization" зі значенням "Bearer api_token", де api_token - це 64-символьний код, який знаходиться в особистому кабінеті на вкладці API Токен
